# API Reference

## Classes
### `SmartAgent`
- **Description:** Core class for creating and managing autonomous agents.
- **Methods:**
  - `run()`: Executes the agent's primary logic.
  - `train()`: Trains the agent on new data.

### `IntegrationManager`
- **Description:** Handles agent integration with external systems.
- **Methods:**
  - `connect(platform)`: Connects the agent to a specified platform.
